Progettazione e sviluppo di un database relazionale ottimizzato per censire e catalogare i dati dei detenuti legati alla criminalità organizzata. 
Il sistema nasce con l'obiettivo di raccogliere ed elaborare le informazioni a livello nazionale, consentendo la generazione di statistiche precise sulla presenza di questa categoria di detenuti in ciascuna regione d'Italia.

Lo sviluppo del progetto si è articolato in tre fasi principali:
- Inizialmente si è proceduto alla formalizzazione dell'idea, traducendola in uno schema concettuale Entità-Relazione (ER).
- Successivamente, è stata eseguita la fase di normalizzazione per definire e ottimizzare lo schema logico.
- Infine, il database è stato implementato a livello di schema fisico tramite il DBMS MySQL, completando il lavoro con la scrittura di alcune query di test per verificarne il corretto funzionamento.
